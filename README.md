# An-lisis-de-embudo-y-retenci-n-de-clientes
Proyeco de Bootcamp, TripleTen para entender en qué etapa del proceso de compra se pierden los usuarios y cómo poder mejorar su retención. Usé SQL para mapear el embudo de conversión completo, identificar los principales puntos de fuga y evaluar la retención de usuarios por cohortes, así como proponer mejoras accionables basadas en los datos.
